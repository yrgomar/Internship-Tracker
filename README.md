# Internship Tracker
![Internship Tracker Screenshot](Screenshot%20(367).png)
A lightweight, browser-based internship application tracker built with vanilla HTML, CSS, and JavaScript.

## Features
- Track applications across 7 status stages (Wishlist → Offer/Rejected)
- Deadline urgency indicators with color coding
- Filter and sort by status, priority, company, or date
- Persistent storage via localStorage — no backend required
- Single HTML file, works fully offline

## Tech
HTML · CSS · Vanilla JavaScript · localStorage API

## Why I built this
Built to track my own internship recruiting process in my sophomore year. Wanted something
fast, offline-capable, and zero-dependency.
